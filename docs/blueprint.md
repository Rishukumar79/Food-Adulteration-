# **App Name**: PureCheck

## Core Features:

- Home Screen: Display categories of food, a quick scan button, and recent tests.
- Image-Based Detection: Upload an image and use the tool in the backend AI model to analyze it, showing possible adulterants, confidence score, and steps for verification.
- Manual Detection Tests: Provide simple at-home tests and scientific explanations for each food item.
- Report Suspicious Food: Allow users to report suspicious food, storing image, product name, shop, user notes, and timestamp in Firestore.
- User Profile: Display user's name, email, past scans, and saved tests.
- Firebase Integration: Use Firebase Authentication, Firestore, Storage, and Cloud Functions for backend functionalities.

## Style Guidelines:

- Primary color: Green (#2ecc71) to represent purity and safety.
- Background color: Light green (#f0f9f4), a desaturated tone of the primary to complement the green, keeps it light.
- Accent color: Blue (#3498db) to highlight important CTAs and information.
- Body and headline font: 'PT Sans', a sans-serif for clear readability and a modern look.
- Use simple, intuitive icons to represent different food categories and actions.
- Clean and modern interface design with clear, easy-to-navigate sections.
- Subtle animations and transitions to enhance user experience.
